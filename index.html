Simulador de datos personales

Este simulador pide al usuario que ingrese sus datos personales, como nombre, edad, sexo, religión y equipo de fútbol.

El simulador utiliza funciones de JavaScript para validar los datos ingresados y asegurarse de que sean correctos.

Una vez que se ingresan todos los datos correctamente, se muestra una alerta con la información ingresada y se pide al usuario que confirme si es correcta.

Si la información es correcta, se agradece al usuario y se almacena la información en un arreglo llamado "respuestas".

El simulador puede ser utilizado para recopilar datos personales de manera segura y eficiente.
